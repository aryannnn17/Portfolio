# Aryan Bhagat Portfolio : https://aryan-bhagat-portfolio.vercel.app/

React + TypeScript portfolio for early-career Software Engineering and Machine Learning roles.

## Stack

- React
- TypeScript
- Vite
- Lucide React icons
- CSS custom properties for light and dark themes

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite, usually `http://localhost:8000`.

## Personalize

- Update content in `src/App.tsx`.
- Replace `public/profile-aryan.webp` and `public/profile-aryan-nav-portrait.webp` with updated profile photos.
- Replace project preview images in `public/` with real screenshots.
- Resume download is served from `public/Aryan_Bhagat_Resume.pdf`.
- GitHub, LinkedIn, email, project links, education, and experience are populated from the resume.

## Build

```bash
npm run build
```
